View someone's DBS check result

Use this service to view someone's DBS check result.

What you will need

  • the one-time share code which the applicant will have been given to you
  • the last six digits of the applicant's certificate number, which they will have been given to you

You can only use the share code once, and you must use it within 21 days.

If you want to view the certificate more than once, you will have to ask the applicant for another share code.

We will let the applicant know when you have viewed their result.
